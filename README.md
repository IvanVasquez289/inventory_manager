# Next.js Product App

Esta es una aplicación fullstack construida con Next.js 14, Prisma y JWT para autenticación. Permite a los usuarios registrarse, iniciar sesión y realizar operaciones CRUD sobre productos.

## Tecnologías

- Next.js 15 
- TypeScript
- Prisma ORM
- TailwindCSS
- JWT para autenticación
- Zustand para manejo de estado global
- Axios para solicitudes HTTP

## Requisitos

- Node.js >= 18
- npm

## Instalación

```bash
npm install
npm run dev
```
